# Desktop Rat Alpha v0.1.0

**Created by Paulina Greybyrd**

A tiny experimental Windows desktop pet! Spawn adorable little rats, drag them around, and watch them scamper across your desktop and windows.

## Setup

1. Extract the ZIP file.
2. Run `Desktop_Creatures.exe`.
3. A tiny rat will appear sitting on the menu!

## Controls

- **Spawn Rat** — Adds a new rat.
- **Clear Rats** — Removes all rats.
- **Always On Top** — Keeps rats above other windows.
- **Exit** — Closes Desktop Rat.
- Drag rats around with mouse. =D

## Features

- 🐀 Multiple rats
- 🪟 Rats run across your desktop windows
- 🖱️ Drag and drop any rat
- 🌳 Pixel art forest menu
- 🖥️ Multi-monitor support

## Notes

- Windows only
- Alpha build
- Rat count is capped for performance
- Feedback and bug reports are appreciated!

## Alpha

Desktop Rat is an early alpha release.

Known limitations:
- Some windows may still be detected as walkable even when hidden behind other windows.
- Performance has been tested with many rats, but extremely large numbers may still reduce performance.

Have fun, and thanks for helping test Desktop Rat!
