# Desktop Rat Alpha v0.1.0

**Created by Paulina Greybyrd**

A tiny experimental Windows desktop pet! Spawn adorable little rats, drag them around, and watch them scamper across your desktop and windows.

## Controls

- **Spawn Rat** — Adds a new rat.
- **Clear Rats** — Removes all rats.
- **Always On Top** — Keeps rats above other windows.
- **Exit** — Closes Desktop Rat.

## Notes

- Windows only
- Alpha build
- Rat count is capped for performance
- Feedback and bug reports are appreciated!

Thank you for trying Desktop Rat! <3
